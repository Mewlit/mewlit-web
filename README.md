## 🚀 Setup

必要なパッケージをインストールします。

```bash
pnpm install
```

## 🛠 Develop

開発用サーバを起動します。

```bash
pnpm dev
```

## 🏗 Build

ウェブサイトをビルドします。

```bash
pnpm build
```

または、静的ファイルを生成します。

```bash
pnpm generate
```

ローカル環境でプレビューを確認するには、以下のコマンドを実行します。

```bash
pnpm preview
```
